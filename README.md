## Курсовая работа №5 DATABASE

## Описание
Данный проект собирает данные по вакансиям от 10-и разных компаний с сайта hh.ru и складывает их в базу данных после чего пользователь может с ними взаимодействовать. К примену найти среднюю зарплату и вакансию по ключевым слову.

## Установка проекта 

Скопировать URL c GitHub и клонировать к себе. Пример кода для клонирования через терминал "git clone https://github.com/вашеимяпользователя/вашпроект.git"

## Установка зависимостей
pip install -r requirements.txt

Запуск программы 'python main.py'

# Основыне функции кода
1. Запрос данных через API HH.ru
2. Создание базы данных 
3. Заполнение базы данных на на основе данных полученных от API без использования файла JSON(тоесть данные сразу поступают в базу данных)
4. Взаимодействие с базой данных, а именно вывод всеч компаний и кол-ва вакансий связанных с этими компаниями, вывод всех вакансий и инфформации о них, вывод средней З/П, вывод вакансий чья З/П выше средней, и поиск вакансий по ключевому слову.
